## 🛠️ How to Create a GitHub Account

1. Go to 👉 https://github.com  
2. Click on **Sign Up**
3. Enter your email, password, and username
4. Verify your email address
5. Complete onboarding steps

---

## ⚙️ Navigate to Developer Settings

After logging in:

1. Open your profile menu (top-right corner)
2. Go to **Settings**
3. Scroll down and click **Developer Settings**

---

## 🔐 Create Personal Access Token (PAT)

1. Inside Developer Settings, click:
   - **Personal Access Tokens**
2. Choose:
   - **Tokens (classic)** or **Fine-grained tokens**
3. Click **Generate new token**
4. Set permissions based on your needs:
   - `repo` → Full control of repositories
   - `workflow` → GitHub Actions access
   - `read:org` → Read organization data
5. Click **Generate token**
6. **Copy and save your token securely (important ⚠️)**

---

## 📌 Important Notes

- Never share your Personal Access Token publicly
- Treat it like a password
- Use environment variables for security in projects
- Fine-grained tokens are recommended for better control
